简介

工具栏主要用于放置按钮等操作的容器，可以放置多个按钮、链接、文本以及分隔符，常用于操作表格、树等控件，BUI中提供了下面几种工具栏：

  1. 一般工具栏：可以任意搭配，包括按钮、文本等工具栏项
  2. 按钮栏：由按钮组成的工具栏,可以用于命令、选中等操作
  3. 分页栏：包括一般的分页栏和数字分页栏

一般工具栏由BUI.Toolbar.Bar类实现，其他工具栏是其子类，BUI.Toolbar.Bar类实现了BUI.Component.UIBase.ChildList扩展，这个扩展实现了列表的所有功能，详情可以参看列表简介

工具栏的选项

工具栏的工具栏项是一个个的控件，工具栏项又以下几种：

类 xclass 描述
通用工具栏项 BUI.Toolbar.BarItem bar-item 通用的工具栏项，可以通过配置模板实现所有任意种类的工具栏项
按钮工具栏项 BUI.Toolbar.BarItem.Button bar-item-button 快速方便的实现按钮功能
文本工具栏项 BUI.Toolbar.BarItem.Text bar-item-text 快速方便的在工具栏上显示文本
空格工具栏项 BUI.Toolbar.BarItem.Spacer bar-item-spacer 方便的实现空格功能
分隔符工具栏项 BUI.Toolbar.BarItem.Separator bar-item-separator 快速方便的在工具栏上生成分隔符

几点说明：

  1. BUI.Toolbar.Bar类的默认xclass是bar-item，使用属性defaultChildClass标示，所以第一个子控件没有声明xclass,xtype，就默认作为bar-item来实现
  2. 由于 defaultChildClass = "bar-item",同时 xclass = defaultChildClass + '-' + xtype，所以当声明xtype : button时，即指定xclass="bar-item-button",其他工具栏项同样实现
  3. spacer类只有一个常用属性width用于指定宽度，separator类，不需要指定任何属性
  4. button类有2个常用属性：btnCls,text,可以通过handler属性快速绑定点击事件
  5. text 类常用text属性指定文本

工具栏嵌套

工具栏可以嵌套使用，一般我们可以使用居左pull-left或者居右pull-left的工具样式定义子工具栏的位置
